# Contenu des Cours : WOLOF

## Unité 0

### Alphabet et Sons
- **A** : comme Alaal (Richesse) (*Voyelle courte.*)
- **À** : comme Àll (Brousse) (*Voyelle longue devant consonne géminée ou nasale.*)
- **B** : comme Benn (Un)
- **C** : comme Ceeb (Riz) (*Toujours prononcé 'tch'.*)
- **D** : comme Dakar (la capitale)
- **E** : comme Emmanuel
- **É** : comme Éréer (Siffler)
- **Ë** : comme Kër (Maison) (*Son central 'eu'.*)
- **F** : comme Fas (Cheval)
- **G** : comme Gént (Rêve)
- **I** : comme Idjig (Hier)
- **J** : comme Jamm (Paix)
- **K** : comme Karat (Carotte)
- **L** : comme Loxo (Main)
- **M** : comme Mburu (Pain)
- **N** : comme Ndox (Eau)
- **Ñ** : comme Ñaar (Deux) (*Nasale palatale 'gn'.*)
- **Ŋ** : comme Caŋ (Lit) (*Nasale vélaire 'ng'.*)
- **O** : comme Oto (Voiture)
- **Ó** : comme Óom (Genou)
- **P** : comme Pom (Pomme)
- **R** : comme Reer (Dîner)
- **S** : comme Safara (Feu)
- **T** : comme Tank (Pied)
- **U** : comme Urus (Or)
- **W** : comme Wanag (Toilettes)
- **X** : comme Xar (Mouton) (*Racle de la gorge (jota).*)
- **Y** : comme Yapp (Viande)

### Pronoms Personnels (Tonic)
- **man** : moi
- **yaw** : toi
- **moom** : lui / elle
- **ñun** : nous
- **yeen** : vous
- **ñoom** : eux / elles

### Syntaxe : Focus Sujet
- **maa** : c'est moi qui... (1S)
- **yaa** : c'est toi qui... (2S)
- **moo** : c'est lui/elle qui... (3S)
- **noo** : c'est nous qui... (1P)
- **yeena** : c'est vous qui... (2P)
- **ñoo** : ce sont eux qui... (3P)
