# Illustrations des Langues Africaines

Ce dossier doit contenir les illustrations pour les 10 langues africaines supportées.

## Fichiers nécessaires :

1. **wolof.png** - Illustration pour le Wolof (Sénégal)
2. **serer.png** - Illustration pour le Serer (Sénégal)
3. **diola.png** - Illustration pour le Diola (Casamance)
4. **peulh.png** - Illustration pour le Pulaar (Sahel)
5. **soninke.png** - Illustration pour le Soninké (Mali, Sénégal)
6. **manjak.png** - Illustration pour le Manjak (Guinée-Bissau)
7. **balante.png** - Illustration pour le Balante (Guinée-Bissau)
8. **mankagne.png** - Illustration pour le Mankagne (Guinée-Bissau, Sénégal)
9. **bambara.png** - Illustration pour le Bambara (Mali)
10. **mandinka.png** - Illustration pour le Mandinka (Gambie)

## Spécifications :

- **Format** : PNG ou JPG
- **Taille** : 800x400px (ratio 2:1) recommandé
- **Style** : Photographies ou illustrations culturelles
- **Thèmes** : Paysages, motifs traditionnels, scènes de vie, instruments, artisanat

## Suggestions de contenu :

### Wolof
- Mosquée de Touba
- Marché Sandaga à Dakar
- Plage de Ngor
- Motifs traditionnels wolof

### Pulaar
- Paysages du Fouta Djalon
- Troupeaux de zébus
- Instruments (hoddu)
- Tissage traditionnel

### Bambara
- Mosquée de Djenné
- Balafon, djembé
- Bogolan (tissu traditionnel)
- Fleuve Niger

### Mandinka
- Fleuve Gambie
- Kora (instrument)
- Marché de Banjul

## Utilisation :

Ces images sont utilisées dans :
- `TargetLanguageStep.tsx` - Sélection de la langue à apprendre
- Écrans d'apprentissage

## Fallback :

En attendant ces images, l'application utilise :
- URLs Unsplash (images temporaires)
- Emoji 🌍 comme placeholder

Consultez `ILLUSTRATIONS_GUIDE.md` à la racine du projet pour plus de détails.
